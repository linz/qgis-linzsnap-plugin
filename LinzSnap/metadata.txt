name=LinzSnap

about=Draws observations, stations, and adjustment vectors calculated 
   using the Land Information New Zealand (LINZ) survey network adjustment
   package SNAP.  SNAP is available at https://www.geodesy.linz.govt.nz/download/snap.
   This plugin requires both SQLite and the spatialite extension to be installed.
   It uses the VectorFieldLayerManager QGIS plugin if it is installed 
   to draw adjustment vectors and error ellipses.
